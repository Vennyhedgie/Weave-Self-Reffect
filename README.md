# Weave-Self-Reffect
A more visible UI element for Weaver's Elite Skill, Weave Self.

-Tracks Weave Self's cooldown and duration, Perfect Weave's duration. 

-Weave Self will be highlighted slightly if you are in combat and the skill is not on cooldown to let you know you should use it.

-Only displays if you have Weave Self equipped.

-Contains two versions, one anchored to the profession bar and another one on top of your health similar to Ranger's F5 skills.
